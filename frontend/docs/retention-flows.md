# Retention Flows (Prototype)

## Vibe-Check Game
- Daily single-prompt interaction with binary answer (`Real` / `Manipulated`).
- Captures user intent signal to support future model training loop.
- Feedback is immediate and lightweight to keep completion friction low.

## Daily City Insight
- One local suggestion card per day (London pilot content).
- Notification-to-card path is mocked in-app for prototype testing.
- CTA focuses on trust and utility: practical time-window and location cues.

## Staycation Mode
- Nearby venues are rateable for co-working and brunch quality.
- Non-travel utility keeps app relevant after booking journey ends.
- Scores are surfaced as quick badges for at-a-glance repeat usage.
