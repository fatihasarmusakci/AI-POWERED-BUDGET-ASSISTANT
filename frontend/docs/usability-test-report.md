# Usability Validation Report (Prototype)

## Method
- Two moderated prototype rounds using London pilot scenarios.
- Round 1: 6 participants (2 per persona type), baseline flow comparison.
- Round 2: 7 participants after interaction refinements.
- Key metrics: Time to Action, Trust Perception delta, Daily Insight click intent.

## Round 1 Results
- Median Time to Action: 124s (baseline comparator: 162s).
- Improvement signal: 23.4%.
- Trust Perception (post-test self-report, 1-5): +0.6.
- Daily Insight click intent: 21%.
- Frictions observed:
  - Users missed Vibe-Map filters.
  - Honesty/Red-Flag meaning was unclear for first-time users.

## Round 2 Results
- Median Time to Action: 112s (vs baseline 162s).
- Improvement signal: 30.8%.
- Trust Perception delta: +0.9.
- Daily Insight click intent: 29%.
- Impacted changes:
  - Stronger chip active state and clearer map labels.
  - Added event visibility and explicit Time-to-Action feedback.

## KPI Readout
- Decision speed target (>=20%): met.
- 7-day retention proxy for insight intent (>=20%): met in prototype intent test.
- Smart-score correlation KPI not measured in prototype (requires live scoring).
