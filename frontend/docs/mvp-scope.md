# VibeCheck MVP Scope Lock

## Included In Prototype
- Persona Engine (3-step onboarding with 3 segments).
- Truth Dashboard with 50/50 official vs user photo comparison.
- AI TL;DR summary card (50 words max style).
- Semantic Score and Honesty Filter as precomputed mock outputs.
- Interactive Vibe-Map layer toggles (Quiet, Party, Metro).
- Daily hooks: Vibe-Check Game, Daily City Insight, Staycation mode.

## Explicitly Excluded In Prototype
- Live Booking and Google Places integrations.
- Production microservice deployment.
- Real NLP/anomaly detection pipeline.
- App store release hardening.

## Acceptance Notes
- Designed for London pilot narrative only.
- Data is mock-driven and deterministic for test sessions.
- Goal is decision-speed and trust UX validation before build-out.
